# HORNET

A 2D platformer game built with Pygame where you play as a brave hornet fighting through 5 islands of enemies and bosses!

## How to Play

```bash
python main.py
```

### Controls
- **Arrow Keys** — Move left/right, jump (up)
- **Space** — Jump
- **Z / X** — Attack with stinger
- **Shift / C** — Dash (after unlocking)
- **S** — Open shop (on island map)
- **Enter** — Confirm / Start
- **ESC** — Back

## The Islands

| Island | Name | Levels | Boss | Power |
|--------|------|--------|------|-------|
| 1 | The Garden Isles | 3 | Wasp King | Double Jump |
| 2 | The Swamp | 4 | Swamp Beetle Lord | Dash |
| 3 | The Crystal Caves | 5 | Crystal Spider Queen | Wall Climb |
| 4 | The Volcano | 6 | Fire Moth | Shield |
| 5 | The Shadow Fortress | 7 | Shadow Hornet | Stinger Upgrade |

## Features

- 25 levels across 5 unique islands
- 5 boss fights with different attack patterns
- 5 unlockable powers (double jump, dash, wall climb, shield, stinger upgrade)
- Coin system — enemies drop coins, bosses drop more
- HP shop — buy extra health with coins
- Save system — progress saves automatically
- Parallax scrolling backgrounds
- Sound effects and music
